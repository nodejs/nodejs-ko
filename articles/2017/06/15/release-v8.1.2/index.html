Node v8.1.2(현재 버전)

  • 작성자: Rod Vagg
  • 원문: Node v8.1.2 (Current)
  • 번역: MJayJang

주요 변경사항

Windows를 제외한 플랫폼에서 네이티브 애드온 컴파일 실패의 원인이었던 8.1.1의 process.release 프로퍼티를 수정했습니다.
Node.js 빌드 프로세스의 수정 사항이므로, 8.1.1에 추가적인 코드 커밋은 없습니다.
