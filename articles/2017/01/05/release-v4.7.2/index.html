Node v4.7.2(LTS)

  • 작성자: Myles Borins
  • 원문: Node v4.7.2 (LTS)
  • 번역: marocchino

이 릴리스에 관하여

이 특별 릴리스는 커밋이 없습니다. 릴리스 이후에 v4.7.1을 위한 추가 플랫폼을
준비하는 동안 릴리스 서버의 tar 배포본이 덮어 쓰여 다른 shasum이 만들어졌습니다.
릴리스에 관한 모호성을 없애기 위해 변경 사항 없이 semver 수 릴리스를
만들었습니다.
