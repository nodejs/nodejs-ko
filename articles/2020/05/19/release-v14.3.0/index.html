Node v14.3.0 (현재 버전)

  • 작성자: Shelley Vohr
  • 원문: Node v14.3.0 (Current)
  • 번역: marocchino

주요 변경 사항

REPL 자동완성 프리뷰 향상

프리뷰 중 <enter>를 누르면 이제 자동 완성된 부분이 포함된 전체 문자열을
평가합니다. <escape>를 누르면 이 동작을 취소합니다.

최상위 레벨 Await 지원

이제 async 함수의 밖에서도 await 키워드를 쓸 수 있습니다.
