Node v12.13.1(LTS)

  • 작성자: Michaël Zasso
  • 원문: Node v12.13.1 (LTS)
  • 번역: Fleta

주요 변경사항

  • Python 3을 사용해 Node.js를 빌드하는 실험적인 지원이 개선되었습니다.
  • ICU 표준시간대 데이터가 2019c 버전으로 업데이트 되었습니다. 이는 브라질의 날짜 오프셋을 수정합니다.
