Node v13.0.1(현재 버전)

  • 작성자: Michaël Zasso
  • 원문: Node v13.0.1 (Current)
  • 번역: Fleta

주요 변경사항

  • deps:
    • Node.js 13.x 버전에서 경고가 발생하는 npm 6.12.0의 버그가 수정되었습니다. (Jordan Harband) #30079.
  • esm:
    • CommonJS 로더와 일치하도록 --es-module-specifier-resolution=node의 파일 확장자 확인 방식이 변경되었습니다. (Myles Borins) #29974.
