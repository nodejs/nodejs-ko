Node v0.12.15(유지보수 버전)

  • 작성자: Rod Vagg
  • 원문: Node v0.12.15 (Maintenance)
  • 번역: Napier

주요 변경사항:

이것은 보안 릴리스입니다. Node.js 사용자는 6월 보안 릴리스 요약 문서에서 패치된 취약점의 자세한 사항을 확인하기 바랍니다.

  • libuv: (CVE-2014-9748) undefined 및 잠재적으로 안전하지 않은 행동을 초래할 수 있는 윈도우 XP와 윈도우 2003의 읽기/쓰기 잠금 구현 버그가 수정되었습니다. 자세한 내용은 https://github.com/libuv/libuv/issues/515 혹은 https://nodejs.org/en/blog/vulnerability/june-2016-security-releases/에서 찾을 수 있습니다.
  • V8: (CVE-2016-1669) V8에서 발견된 잠재적 버퍼 오버플로 취약성이 수정 되었습니다. 자세한 내용은 CVE 혹은 https://nodejs.github.io/nodejs-ko/articles/2016/06/16/june-2016-security-release/에서 찾을 수 있습니다.
